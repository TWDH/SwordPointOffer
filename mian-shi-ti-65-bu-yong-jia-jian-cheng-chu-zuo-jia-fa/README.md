# 面试题65：不用加减乘除做加法

## 方法一：二进制算法

1. 进行非进位和，只单纯相加数字并不计算进位。**异或**运算
   * `0 + 0 = 0`, `1 + 1 = 0`, `1 + 0 = 1`。
2. 计算进位carry。**与**运算+**左移**一位
   * 不进位：`0 + 0`，`0 + 1`， `1 + 0` 
   * 进位：1 + 1
3. 将前两步相加，直到没有进位

![](../.gitbook/assets/image%20%288%29.png)

```java
class Solution {
    public int add(int a, int b) {
        int sum, carry;

        while(b != 0){
            //0.提前计算进位
            carry = (a & b) << 1;
            //1.将两数相加，不进位。异或
            a = a ^ b;
            b = carry;
        }
        return a;
    }
}
```
